What is it?

Bumping bunnies is a remake of the classic game Jump'n'Bump.
Play against friends on Android or on the PC. WLAN is supported for cross-platform games. Additionally Bluetooth is supported for android.

PC

Download

  • bumping-bunnies-pc.jar
    • Bumping bunnies requires Java 8. Go here to download the newest JRE.

    Installation instructions:
    • Install the JRE if necessary.
    • Double click bumping-bunnies-pc.jar
    • Or start via command line: java -jar bumping-bunnies-pc.jar

    Supported platforms

    All platforms which support Java are supported.

    Additional information

    Bumping bunnies is open source. You can the source on Github.
